##  Installation & Setup

**Clone repo & install dependencies**:


```bash
git clone https://github.com/muhammedajmalkt/royalEnfield-Sahyadri.git
cd royalEnfield-Sahyadri

npm install

```



**Run the development server**:

```bash
npm run dev
```
**Open in your browser**:
```bash
http://localhost:3000

```
